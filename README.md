## Getting Started

```bash
# Clone repository
git clone <repository-url>
cd xstream/web-app

# Install dependencies
npm install

# Configure environment
cp .env.example .env.local

# Setup database
npm run db:push
npm run db:seed

# Start development server
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to explore xStream.
